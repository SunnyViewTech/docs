# 肩幅の測定

## 測定の目的

最終的なキャラクターの体型比率がモーションキャプチャ参照モデルと異なる場合、合掌のように手が近づくポーズでは、手が離れすぎたり交差したりする可能性があります。適切な肩幅を設定することで、これらの問題を軽減できます。

:::warning ご注意

リアルタイムモーションキャプチャでは、手を組むなど体の部位同士が接触するアクションは技術的に難しく、すべてのケースで完璧に再現できるとは限りません。ご了承ください。

:::

## 測定方法

測定を開始する前に、SAYA がモデルと同期していることを確認してください。このプロセスはモデルのポーズを観察することに依存しています。

プログラム下部のボタンをクリックして測定を開始します。

![](../img/2026_01_04_15_06_262.png)

測定中は、胸の前で指先を近づけてください。手をきつく押し付けないでください。トラッキングが混乱し、精度に影響を与える可能性があります。

![](../img/1765619894422.jpg)

測定が開始されると、プログラムは自動的に肩幅を継続的に調整します。モデルをよく観察し、手首がちょうど触れた時に頭をうなずいて確認してください。

または、手首が触れた瞬間に画面に表示される肩幅値をメモし、設定で手動入力することもできます。

![](../img/2026_01_04_15_39_17.png)

同じボタンを再度クリックすることで、いつでも測定をキャンセルできます。

測定完了後、より良い結果を得るために肩幅をさらに微調整することができます。

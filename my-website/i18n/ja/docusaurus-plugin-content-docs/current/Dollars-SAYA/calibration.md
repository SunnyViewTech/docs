# キャリブレーション

## キャリブレーションの目的

iOS デバイスを配置する際に前後に傾いていると、キャプチャされたキャラクターもそれに応じて傾く可能性があります。キャリブレーションはこの問題を修正するのに役立ちます。

また、個人の顔の特徴によって、表情精度に影響を与えるベースラインのずれが生じる場合があります。キャリブレーションにより、動的な表情をより正確に検出できるようになります。

## キャリブレーション方法

プログラム下部のボタンをクリックしてキャリブレーションを開始します。

![](../img/2026_01_04_15_06_26.png)

キャリブレーション中は、体と表情をリラックスさせ、腕を自然に垂らし、正面を向いてください。

カウントダウンが終了し、「キャリブレーション完了」のメッセージが表示されたら、通常のモーションキャプチャを開始できます。

同じボタンを再度クリックすることで、いつでもキャリブレーションをキャンセルできます。
